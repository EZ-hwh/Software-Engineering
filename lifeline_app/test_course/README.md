# Life Line 课程界面

使用vue搭建的前端：

## 环境配置

- 和ncr之前的一样 通过yarn

## 运行

```bash

# 1. Open command line and go to this folder

# 2. Install dependencies. Make sure yarn is installed: https://yarnpkg.com/lang/en/docs/install
yarn install

# 3. Run following command to start the development server
yarn dev
```

## 结构

```
-public

	design

```
